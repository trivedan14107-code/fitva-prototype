# FITVA — Your AI Fitness Companion

## How to Run (3 steps)

### Step 1: Open Terminal in this folder
Right-click the `fitva-app` folder → "Open in Terminal"
(or open VS Code, then press Ctrl+` to open terminal)

### Step 2: Install dependencies
```
npm install
```
Wait for it to finish (takes ~30 seconds).

### Step 3: Start the app
```
npm run dev
```
This opens the app at **http://localhost:3000** in your browser.

---

## ⚠️ DO NOT run `node App.jsx` — that does NOT work for React apps.
React needs a build tool (Vite) which is included in this project.

---

## Requirements
- Node.js 18+ (you already have this)
- npm (comes with Node.js)

## Features
- Splash screen with animations
- Login / Signup authentication
- 4-step fitness onboarding
- Avatar creation & evolution
- Workout tracking with timer
- AI food scanner
- Water intake tracker
- Community feed with Boost/Challenge
- Challenge system with XP rewards
- Supplements recommendations
- Progress dashboard with charts
- Profile management
